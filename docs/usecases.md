# Mangala Wallet App Use Cases

Based on the main-app-flows.mermaid diagram, here are the key use cases for the Mangala wallet application:

## Authentication & Security
1. Unlock wallet - Enter PIN or use biometric authentication to access wallet
2. Set up new PIN - Create and confirm a secure PIN for wallet access
3. Enable biometric authentication - Set up fingerprint/face ID for quick access
4. Recover forgotten PIN - Reset PIN through recovery process

## Wallet Management
5. Create new wallet - Generate a new wallet with seed phrase
6. Backup wallet - Save recovery phrase securely
7. Restore wallet - Import existing wallet using recovery phrase
8. Import account by private key - Add accounts using private keys
9. Import account by certificate - Add accounts using key certificates
10. Manage multiple accounts - Switch between and organize different accounts

## Token Operations
11. View portfolio - Check total balance and asset allocation
12. Send tokens - Transfer tokens to other addresses
13. Receive tokens - Generate QR codes and share addresses for receiving
14. Swap tokens - Exchange one token for another
15. View transaction history - Browse, filter, and search past transactions

## NFT Management
16. View NFT collection - Browse owned NFTs across different chains
17. Import NFTs - Add NFTs to collection by contract details
18. Send NFTs - Transfer NFTs to other addresses
19. View NFT details - Check properties and metadata

## Antelope-Specific Features
20. Create Antelope account - Set up new EOS/Telos/WAX accounts
21. Manage account permissions - Create, update, link/unlink authorities
22. Manage resources - Stake/unstake CPU, NET, buy/sell RAM
23. Rent resources - Use REX or PowerUp for temporary resources
24. Create/manage multisig proposals - Collaborative transaction management
25. Backup Antelope accounts - Export keys and certificates

## DApp Integration
26. Browse Web3 sites - Access decentralized applications
27. Connect to dApps - WalletConnect integration
28. Sign transactions - Approve dApp transactions
29. Sign messages - Authenticate with dApps
30. Switch chains - Change networks for dApp compatibility

## Crypto Payments
31. Pay with crypto - Make payments using various cryptocurrencies
32. Select payment method - Choose token and network for payment
33. Manage allowances - Approve spending limits for smart contracts

## Settings & Preferences
34. Change theme - Switch between light/dark modes
35. Select language - Choose preferred app language
36. Set fiat currency - Configure display currency
37. Manage networks - Add custom RPC endpoints
38. Configure notifications - Set alert preferences
39. Manage contacts - Save frequently used addresses

## Cold Wallet Features
40. Sync accounts - QR-based account synchronization
41. Offline transaction signing - Sign transactions without internet
42. QR transaction flow - Complete transactions via QR codes

## Support & Information
43. Access help center - Browse FAQs and guides
44. Contact support - Get assistance
45. Share app - Refer friends to the wallet
